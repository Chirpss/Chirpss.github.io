Your privacy is important to us. This policy details how your data is handled.

1. Responsible Entity (Controller)

The entity responsible for data processing (the "Controller") is Chirpss.

For information regarding the Controllership or data protection inquiries, please use the following dedicated contact channels:

  • Privacy Inquiries (DPO/Data Subject Rights): Privacy@chirpss.my.to
  • Legal Inquiries (Controller): Legal@chirpss.my.to

2. Data Processing & Game Operation

Processing is based on Art. 6(1) lit. b GDPR (contract performance) and Art. 6(1) lit. f GDPR (legitimate interest: functional operation).

  • Player Names: Names (e.g., in Imposter) are stored temporarily in browser Session Storage or in a Firebase Realtime Database for the duration of a single game session only. This data is deleted automatically when the game ends or the session expires.
  • Persistent IDs: Games (e.g., MostLikely, WhoAmI) store a unique, anonymous ID (p_abcdef123) in your Local Storage to recognize your session if the page refreshes. This ID is solely for game function and contains no personal information.
  • Prompts/Ratings: Data submitted for statistical features (e.g., 10/10 but ratings) are anonymized and stored on a Firebase Realtime Database to calculate group statistics. This data does not contain personal identifiers.

3. External Tools & Data Sharing

  • Firebase Realtime Database: Used for all Multi-Device Games to synchronize game states and temporary player data. Data is hosted in the west of Europe..
  • Google Analytics / Tag Manager: Used for analytical purposes. Tracks website usage to improve our service. IP anonymization is enabled.

4. Your GDPR Rights

As a data subject, you have rights including access, rectification, erasure, restriction of processing, data portability, and objection regarding your personal data.

  • To Exercise Rights: Please contact the Controller via the email provided in Section 1 (Privacy@chirpss.my.to).
  • Right to Lodge a Complaint: You have the right to lodge a complaint with a supervisory authority (Data Protection Authority) in your respective Member State.
